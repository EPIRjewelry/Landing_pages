# Hydrogen Landing (EPIR)

Minimalny landing oparty o React + Vite + Tailwind.

## Wymagania
- Node.js 18+

## Konfiguracja
Skopiuj `.env.example` do `.env` i uzupełnij wartości:
- `VITE_STORE_DOMAIN`
- `VITE_STOREFRONT_TOKEN`
- `VITE_STOREFRONT_API_VERSION` (opcjonalnie)
- `VITE_COLLECTION_HANDLE` (opcjonalnie, aby wymusić konkretną kolekcję)

## Uruchomienie
- `npm install`
- `npm run dev`

## Build
- `npm run build`
- `npm run preview`
